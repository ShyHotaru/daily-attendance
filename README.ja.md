# Hotaru's Daily Attendance

[English](README.md) | [한국어](README.ko.md) | **日本語**

Windows ログイン時に、指定したゲームの出席ページを **1日1回だけ** 自動でブラウザに開く小さなユーティリティ。

## 特徴

- Windows スタートアップに登録され、ログイン時に自動で実行
- **1日1回のみ実行** — ログアウトして再ログインしても重複して開かない
- シンプルな GUI でゲームの選択とカスタム URL の追加が可能

## インストール

1. [Releases](../../releases) から最新の `HTR_DA.exe` をダウンロード
2. `HTR_DA.exe` を実行 → 開きたいゲームをチェック → **Install / Update** をクリック
3. 次回の Windows ログインから自動で実行されます

## カスタムゲームの追加

GUI 上で **+ Add Custom Game** ボタンをクリックし、名前と URL を入力してください。

追加したカスタムゲームを削除するには、該当項目の横にある **×** ボタンをクリックします。

## アンインストール

- GUI を起動して **Uninstall** ボタンをクリック、または
- コマンドラインから: `HTR_DA.exe --uninstall`

## ファイルの配置

| 用途 | パス |
|------|------|
| 設定 | `%APPDATA%\HTR_AttendanceAuto\config\` |
| 1日1回フラグ | `%APPDATA%\HTR_AttendanceAuto\flags\` |
| インストール済み実行ファイル | `%APPDATA%\HTR_AttendanceAuto\bin\HTR_DA.exe` |
| スタートアップ登録 | `%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup\HTR_daily_attendance.vbs` |

## 動作の仕組み

Windows にログインすると、プログラムはまず **今日すでに出席ページを開いたかどうか** を確認します。

- まだ開いていなければ → 設定された出席ページを既定のブラウザで開きます。
- すでに開いていれば → 重複実行せずに即座に終了します。

バックグラウンドサービスやタスクスケジューラを使わず、シンプルなファイルベースの方式で「1日1回」の動作を保証しています。

## ポリシーに関するお知らせ

本ツールは既定のブラウザで出席ページを開く動作のみを行い、ユーザーに代わってログイン・クリック・データ送信などは一切**行いません**。

それでも、特定のページを本ツールで開くことが当該サイトの利用規約や運営ポリシーに抵触すると思われる場合は、[Issue からご連絡ください](../../issues)。該当ゲームをデフォルトカタログから削除いたします。
